# DropHunt

## Setup

```
bun i
```

## Run website

```
bun web
```

## Run API

Edit `.dev.vars` and put:

```
RONIN_TOKEN=
TOKEN=
```

API requires deve of Drophunt access, or figure out how to create Ronin workspace and set the token on api/client yourself.

Then run with:

```
bun api
```

## Run Telegram mini app

```
bun app
```
